type()
The type() function is used to get the data type of a value. It is especially useful for beginners because it can easily tell you the data type of a value.
In image 1, we used the type() function on the string “Hello World!”. The result was str, which is short for string. Therefore, “Hello World!” is a string. In image 2, we also used the type() function on “200”. While it may look like an integer, it has quotations and so it is a string, as the function tells us. Finally, image 3 shows an example of an integer. When we do type(56), we get int because 56 is an integer and there are no quotation marks around it.
